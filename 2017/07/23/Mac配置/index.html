Mac配置

Posted on 2017-07-23

前言

做个备忘，将常用的mac软件做成清单。

软件清单

系统工具

  • Jitouch      Mac上自定义触摸板手势的软件，设定了等等操作。
  • Windows Tidy      Mac上快速分屏软件，可以实现windows的分屏功能。（小小吐槽，为啥Mac这系统连个自带的分屏功能都没有，windows下把窗口拖到屏幕边框就会提示你分屏，这是对于多线程狗来说多么实用的功能）
  • HyperSwitch      Mac上快速切换应用的软件，可以实现windows的ctrl+tab快速切换窗口的功能。
  • Teamviewer      著名的远程桌面软件，可以让我在何时何地连接寝室的计算机。
  • Mou      Mac上Markdown编辑器，实时显示，外观好看。
  • IINA      Mac上强大的本地视频播放器，感觉和windows的Potplyer有的一拼。
  • IRUVE      Mac上自动更换壁纸的软件，壁纸源很广，有著名的cuto等，质量颇高。

生产力工具

  • Xcode      Mac上进行C++编程必须的软件，集合编辑器，编译器，调试工具等等于一体。
  • Vim      编辑器之神，因为在台式机上编程是在linux系统下的，使用的编辑器便是Vim，所以在这里也装上了，各种插件美滋滋。
  • Anaconda      Mac上进行Python编程所需的软件，（也不是必须的），其中较为常用的是Juypter。
  • Sublime      Mac上一个好用的编辑器，主要用来进行latex文档的编辑。
  • Texlive      Mac上使用latex所需的软件。为了省心，我直接把latex全部包都装好了。
  • Skim      打开速度超快的pdf阅览器，在windows上打开pdf的速度也是秒杀adobe和福昕的。

科学工具

  • ShadowsockX Mac上科学上网的工具，科研狗看论文伤不起。

娱乐软件

  • Steam 坑钱平台。
